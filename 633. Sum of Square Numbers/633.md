# Sum of Square Numbers #

Given a non-negative integer c, your task is to decide whether there're two integers a and b such that a^2 + b^2 = c.

**Example 1:**
```
Input: 5
Output: True
Explanation: 1 * 1 + 2 * 2 = 5
```
**Example 2:**
```
Input: 3
Output: False
```

---

## 解题思路 ##
提示：一道基本的查找题，定两头指针，每个指针只有一个移动方向。
